1. Introduction

The AI Smart Interview System is a modern web application that allows students and job seekers to practice real-world job interviews using Artificial Intelligence. The system generates intelligent interview questions based on the candidate's role and resume, evaluates their spoken answers in real-time, and provides detailed feedback on performance metrics like technical knowledge, communication, and confidence.

The platform supports Voice-based answering (Speech-to-Text), AI-generated questions via OpenAI GPT-3.5, and adaptive difficulty that changes based on candidate performance.

2. Objectives

  • Build an AI-powered platform that simulates real job interview experiences.
  • Generate role-specific interview questions dynamically using OpenAI API.
  • Enable voice-based answer submission using Web Speech Recognition API.
  • Analyze and score candidate answers across multiple parameters.
  • Maintain a full interview history for tracking progress over time.
  • Extract skills from uploaded resumes and personalize questions accordingly.
  • Provide adaptive difficulty that adjusts based on candidate performance.

3. Problem Statement

Traditional interview preparation relies on books, mock interviews, or expensive coaching. There is no affordable, accessible, and intelligent platform that can:

  • Simulate real interview conditions with dynamic AI questions.
  • Accept spoken voice answers like a real interviewer.
  • Give instant, detailed, multi-dimensional feedback.
  • Adapt question difficulty based on candidate performance.

The AI Smart Interview System solves all these problems in one integrated web platform.

6. Module Descriptions

🔐 Authentication

  • User Registration with bcrypt password hashing
  • JWT-based secure login
  • Protected routes via middleware

🎤 Interview Room

  • Role: Software Dev / HR / Managerial
  • Difficulty: Easy / Medium / Hard
  • Voice answer via Web Speech API
  • Adaptive difficulty engine

🧠 AI Analysis

  • Overall / Technical / Communication / Confidence Score
  • Strengths & Weaknesses analysis
  • Suggested Answer + General Feedback

📄 Resume Upload

  • PDF upload via Multer
  • Text extraction using pdf-parse
  • Skills saved & used to personalize questions

📜 Interview History

  • View all past sessions
  • See Role, Score, Questions count
  • Delete individual records

📊 Dashboard

  • Interview statistics overview
  • Progress tracking
  • Quick navigation to all features
